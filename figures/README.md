# Figures Directory

## Overview  
This folder contains all visual outputs generated during the exploratory analysis and modeling phases of the project.

---

## Contents  

The figures include:

- Data distribution plots  
- Correlation heatmaps  
- Feature importance visualizations  
- Model performance comparisons  
- ROC curves and evaluation plots  

---

## Purpose  
These visualizations are used to:
- Understand data patterns  
- Support model interpretation  
- Communicate results clearly  
- Provide evidence for feature relevance  

---

## Notes  
- All figures are generated from notebooks  
- Images are used in analysis and documentation  
- Only relevant and final versions of plots are stored here
